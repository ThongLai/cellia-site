# Cellia's Personal Website

Personal website for my girlfriend

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **UI Components**: shadcn/ui
- **Animations**: Framer Motion
- **Database**: Supabase
- **Icons**: Lucide React

## Pages

- **Home** (`/`) - Hero section with avatar, featured projects, and skills
- **About** (`/about`) - Professional background, education, skills, and awards
- **Portfolio** (`/portfolio`) - Project grid with filtering and detail pages
- **Blog** (`/blog`) - Article listings with category filters
- **Gallery** (`/gallery`) - Image gallery with upload functionality
- **Contact** (`/contact`) - Contact form and information

## Content
All content is stored in Supabase and can be managed through the database or by an admin interface.
